# Public-Safe Strategy

This repository is designed to be shared publicly without exposing the complete private optimizer implementation.

## What Is Public

- system architecture
- API contracts
- typed schemas
- runnable demonstration pipeline
- tests
- CI workflow
- Docker support
- documentation

## What Is Intentionally Abstracted

- proprietary learned policies
- private scoring weights
- production model routing strategy
- private A.R.E.S backend payload details
- production memory/retrieval stores
- deployment secrets and runtime data

## Why This Matters

A strong portfolio repo should show engineering ability without giving away every implementation detail. This project presents the system design and integration surface while keeping the private competitive logic protected.
